# 字符串整数转换

对于一个字符串需要将其转为整数(1-index)，要用的时候能及时调出该整数，给出整数也要返回相应字符串。



```c++
struct strconint
{
	unordered_map<string,int> um;
	vector<string> v;
	string int2str(int x)
	{
		return v[x-1];
	}
	int str2int(string s)
	{
		if(um.find(s)==um.end())
		{
			v.pb(s);
			um[s] = (int)v.size();
			return (int)v.size();
		}
		else
			return um[s];
	}
} si;
```
